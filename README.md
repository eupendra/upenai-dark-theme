# Upenai Dark Theme

## Build

```bash
npx @vscode/vsce package
```

## Install

### VS Code
```bash
code --install-extension upenai-dark-theme-0.0.1.vsix
```

### Cursor
```bash
cursor --install-extension upenai-dark-theme-0.0.1.vsix
```

Or manually: Open Command Palette → `Extensions: Install from VSIX` → Select the `.vsix` file
